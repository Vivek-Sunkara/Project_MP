Graphical solution LP

The graphical solution to Linear Programming (LP) is used to visually solve optimization problems with two decision variables.

Understand Feasibility: Identifies the feasible region satisfying all constraints.

Find Optimal Solutions: Determines maximum or minimum values of the objective function at the region's vertices.

Simplify Concepts: Provides a clear, geometric representation for small-scale problems.

LP - Simplex Method

The Simplex Method is a step-by-step algorithm used to solve Linear Programming (LP) problems, which aim to maximize or minimize a linear objective function (like profit or cost) while satisfying constraints.

How It Works:

Set up equations for the objective and constraints.

Organize them into a table (Simplex tableau).

Move systematically between feasible solutions to improve the objective.

Stop when the best solution is found.

Transportation

The Transportation Problem is a type of Linear Programming problem where the objective is to determine the most cost-effective way to transport goods from multiple suppliers to multiple consumers.

Methods Used:

1. Northwest Corner Method: A basic feasible solution is obtained by starting at the top-left (northwest) corner of the cost matrix and moving either right or down.

2. Least Cost Method: This method selects the cell with the smallest cost and allocates as much as possible to that cell, then moves to the next smallest cost cell.

3. Vogel's Approximation Method (VAM): This method calculates penalties for not using the least cost route and allocates to the route with the highest penalty.

4. MODI Method: The Modified Distribution Method (MODI) is used to find the optimal solution by improving the initial feasible solution obtained by the above methods.
